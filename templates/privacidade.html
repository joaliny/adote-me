🔒 Política de Privacidade

Adote-Me - Proteção de Dados Pessoais

← Voltar para o Cadastro

1. Introdução

Esta Política de Privacidade descreve como o Adote-Me coleta, usa, armazena e protege suas informações pessoais, em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei 13.709/2018).

2. Dados que Coletamos

2.1 Dados Pessoais

  • Nome completo
  • CPF (para processo de adoção)
  • Endereço de e-mail
  • Telefone
  • Endereço residencial
  • Data de nascimento

2.2 Dados de Uso

  • Endereço IP
  • Tipo de navegador
  • Páginas visitadas
  • Tempo de permanência
  • Interações com a plataforma

3. Como Usamos Seus Dados

  • Processar adoções: Validar e gerenciar processos de adoção
  • Comunicação: Enviar atualizações sobre seus processos
  • Melhorar serviços: Analisar uso para melhorias
  • Segurança: Prevenir fraudes e atividades suspeitas
  • Conformidade legal: Cumprir obrigações legais

4. Compartilhamento de Dados

Seus dados podem ser compartilhados apenas nas seguintes situações:

4.1 Com Outros Usuários

  • Doadores: Informações de contato para adotantes aprovados
  • Adotantes: Dados básicos do doador para contato

4.2 Com Terceiros

  • Parceiros veterinários: Para agendamento de consultas
  • Autoridades: Quando exigido por lei
  • Provedores de serviço: Hospedagem, e-mail marketing

⚠️ Atenção: Nunca compartilhamos seus dados com fins comerciais para terceiros não relacionados aos serviços de adoção.

5. Armazenamento e Segurança

5.1 Medidas de Segurança

  • Criptografia de dados sensíveis
  • Acesso restrito a funcionários autorizados
  • Backups regulares
  • Monitoramento de segurança 24/7

5.2 Tempo de Retenção

Mantemos seus dados apenas pelo tempo necessário para:

  • Cumprir finalidades legítimas
  • Atender obrigações legais
  • Resolver disputas
  • Manter registros de adoção

6. Seus Direitos

Conforme a LGPD, você tem direito a:

  • Confirmação: Saber se seus dados são processados
  • Acesso: Solicitar cópia de seus dados
  • Correção: Atualizar dados incompletos ou errados
  • Anonimização: Solicitar anonimização de dados
  • Eliminação: Solicitar exclusão de dados
  • Portabilidade: Receber dados em formato legível
  • Revogação: Revogar consentimentos

7. Cookies e Tecnologias Similares

Utilizamos cookies para:

  • Lembrar suas preferências
  • Melhorar performance do site
  • Analisar uso da plataforma
  • Personalizar conteúdo

Você pode controlar cookies através das configurações do seu navegador.

8. Menores de Idade

Não coletamos intencionalmente dados de menores de 18 anos. O processo de adoção requer maioridade civil.

9. Alterações nesta Política

Podemos atualizar esta política periodicamente. Notificaremos sobre mudanças significativas através de:

  • E-mail registrado
  • Notificação na plataforma
  • Atualização da data de revisão
